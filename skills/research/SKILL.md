# Research

## Description
Web research and information gathering.

## Instructions
1. Formulate effective search queries
2. Search using webSearch
3. Fetch relevant pages with fetchPage
4. Synthesize into a clear summary with sources

## Usage
Use when the task requires information from the web: looking up docs, finding solutions, current events, API references.

## Technical Implementation
Uses Tavily API for search, direct HTTP fetch for page content.
