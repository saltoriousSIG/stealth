# Research Skill

Web research and information gathering. Search the web, fetch page content, and synthesize findings into useful summaries.

## Triggers

- Search for
- Research
- Find information
- Look up
- What is
- How to

## Model

Uses: local-fast

## Tools

- webSearch
- fetchPage
- readFile
- writeFile

## Execution Pattern

1. Understand the research question or topic
2. Formulate effective search queries
3. Search the web using Tavily
4. Fetch and analyze relevant pages
5. Synthesize findings into a clear summary
6. Cite sources when appropriate
