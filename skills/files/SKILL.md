# Files Skill

Manage file system operations. Read, write, search, and organize files and directories.

## Triggers

- Read file
- Write file
- Create file
- List files
- Find file
- Show contents
- Save to file

## Model

Uses: local-fast

## Tools

- readFile
- writeFile
- listDirectory

## Execution Pattern

1. Understand the file operation needed
2. Verify paths and permissions
3. Execute the file operation
4. Confirm success or report errors
5. Summarize what was done
